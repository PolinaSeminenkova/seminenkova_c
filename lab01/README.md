Задание 8. Переполнение
Запуск: ./task8
Программа прибавляет 1 к максимальному значению int (2147483647). Ничего вводить не нужно.

Объяснение: Максимальное значение для 32-битного int = 2147483647. При прибавлении 1 происходит переполнение — старший бит становится 1, число интерпретируется как отрицательное (минимальное значение int = -2147483648).

Задание 14. Исследование памяти
Запуск: ./task14
Программа выводит адреса трёх переменных int и разницу между ними. Ничего вводить не нужно.

Объяснение: Переменные располагаются в стеке. Разница между адресами показывает, сколько байтов памяти занимает каждая переменная с учётом выравнивания. Обычно разница равна 4 (для int) или больше из-за выравнивания компилятором.
