### Ответы на вопросы

Как объявить структуру в языке C?
struct Student {
    char name[50];
    int age;
    float ball;
};
Что такое вложенные структуры и зачем они нужны?
Это структура внутри другой структуры. Нужны для группировки связанных данных (например, дата рождения внутри структуры человека).

Как передать структуру в функцию?

по значению: void func(struct Student s)

по указателю: void func(struct Student *s)

Можно ли создавать массивы структур?
Да: struct Student group[30];

Как обратиться к элементу массива структур?
Через индекс и точку: group[0].name

Можно ли использовать enum внутри структуры?
Да: struct Student { enum Course course; };
